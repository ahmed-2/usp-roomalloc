# Constraints

property `floor`
true for [4, 5, ... 21]

property `male_floor`
true for [8, 10, 16, 19]

property `female_floor`
true for [7, 14, 18, 20]

property `mixed_floor`
true if not in `male_floor` and not in `female_floor`

property `room`
    true if floor_id is a valid floor
    ture if room_id is a valid room_number
